# Not just dockerfiles!

- CLI tools: Dockerfiles and sometimes associated packages for CLI utilities. Meant to be run with $docker run$ etc. Most are publicly available, some I created (with help from Claude).
- Webservers: Dockerfiles for tools that run as live web pages, such as primer3plus. Meant to be run on a VM and served via ports forwarded over ssh.
- Dev Containers: More devcontainer.json files than dockerfiles, though those are included when necessary. For data analysis or development in python, R, etc. Also using suites of packages for specific applications.
- Standalone HTML: Javascript apps designed to be shared as html files and run in the browser without installation. Extremely portable and convenient.
